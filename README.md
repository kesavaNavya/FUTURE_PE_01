# FUTURE_PE_01
AI-powered YouTube video creation using prompt chains. Generates titles, scripts, thumbnails, and a fully edited video using modern AI tools.

## Overview
This project showcases how **prompt chaining and AI tools** can be used to create a complete YouTube video from scratch. It covers title generation, script writing, thumbnail creation, and AI-based video editing.

## Tools Used
- ChatGPT  
- Pictory / InVideo  
- Canva  
- DALL·E / Midjourney  
- Notion

## Workflow
1. Select a YouTube niche  
2. Generate titles using prompt chains  
3. Write a long-form video script (800–1500 words)  
4. Create thumbnail text and AI visuals  
5. Edit and produce the final video using AI tools

## Deliverables
- Prompt chain documentation  
- Full video script  
- Thumbnail text and AI-generated image  
- Final AI-edited video
